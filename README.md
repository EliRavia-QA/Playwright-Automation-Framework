# 🚀 Playwright & Pytest Automation Framework

Welcome to my professional automation project! This framework is designed to demonstrate high-level testing capabilities for both Web UI and API layers.

## 🛠 Features
- **Hybrid Testing:** Full coverage for Web UI (SauceDemo) and REST API (JSONPlaceholder).
- **Page Object Model (POM):** Clean, modular, and maintainable architecture.
- **Advanced Reporting:** Integrated **Allure Reports** including:
  - Failure screenshots 📸
  - Browser console logs 🗒️
  - Dynamic environment properties.
- **Data-Driven:** Configuration managed via `.ini` files for easy updates.

## 🏗 Project Structure
- `pages/`: Page Objects representing web elements and actions.
- `tests/`: Organized test suites for Login and API endpoints.
- `utils/`: Reusable utilities like Config Readers and API managers.
- `conftest.py`: Centralized fixtures for Browser and API contexts.
